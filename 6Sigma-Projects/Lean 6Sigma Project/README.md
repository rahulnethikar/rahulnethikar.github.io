# Lean Six Sigma: Premium Tech Support Operations Excellence
**Organization:** Selectsys India | **Client:** US Tech Support
**Project Period:** February 2012 – May 2012 (3 Months)

---

## 🚀 Project Overview
This repository contains the complete documentation suite for a **Lean-driven process improvement initiative** focused on transforming the Premium Tech Support operations at Selectsys India. The project was executed to resolve critical gaps in Customer Satisfaction (CSAT), Agent Utilization, and Service Quality.

### Key Focus Areas:
*   **Chat Support (L1):** Standardizing communication and professionalism.
*   **Voice Support (L2):** Improving reach rates and technical resolution.
*   **Remote Troubleshooting (LMI):** Optimizing the "Restart Protocol" to prevent session abandonment.
*   **Operational Governance:** Enforcing biometric-driven discipline and real-time utilization monitoring.

---

## 📈 Executive Summary of Results
The project achieved a significant turnaround in key operational metrics over the 90-day execution period:

| Metric | Baseline (Feb 2012) | Final (May 2012) | Improvement |
| :--- | :--- | :--- | :--- |
| **CSAT Score** | ~75% | **95.2%** | +20.2% |
| **Agent Utilization** | ~30% | **90.3%** | +60.3% |
| **Reach Rate** | ~30% | **76.3%** | +46.3% |
| **Average QA Score** | ~78% | **91.3%** | +13.3% |

## 🏗️ Project Governance & Methodology
The project followed a structured **DMAIC (Define, Measure, Analyze, Improve, Control)** framework, emphasizing Lean principles for waste reduction and operational efficiency.

*   **Process Owner:** Operations Manager (OM)
*   **Execution Lead / Process SME:** Rahul Nethikar (Assistant Manager)
*   **Stakeholders:** Chat Support AM, Voice TL, Quality Assurance (QA), and HR.

## 🛠️ Key Improvements Implemented
1.  **Restart Protocol Disclaimer:** A mandatory script requiring verbal/chat consent before restarts to prevent premature survey triggers.
2.  **Preformatted Response Library:** 50+ standardized chat responses to ensure professional English communication.
3.  **Real-Time Utilization Monitoring:** 2-hourly Cudatel VOIP reports to minimize "Idle" and "Aux" time leakage.
4.  **Reach Tracker:** Mandatory 3-attempt callback policy for all technical escalations.
