## CURRENT STATE ANALYSIS
*   **NPS Distribution:** Promoters (28%), Passives (59%), Detractors (13%). *Correction: Usually NPS = P-D. If P=28, D=13, NPS=15%.*
*   **Detractor Drivers:**
    1.  Long hold times (35%)
    2.  Repeat calls for same issue (30%)
    3.  Lack of empathy/Mechanical responses (25%)
    4.  Technical knowledge gap (10%)
*   **AHT Trend:** Stable at 6.5 mins, but high variability across agents (3 mins to 12 mins).
*   **FCR Correlation:** Low FCR is directly leading to lower NPS scores from repeat callers.
