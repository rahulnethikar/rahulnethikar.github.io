# Project: Improving Centre NPS Using Time and Motion Study

## 🌑 Black Belt Project Overview (Early 2011)

This project focused on improving the Net Promoter Score (NPS) for the Verizon DSL Technical Support queue at Mahindra Satyam BPO. By leveraging a granular **Time and Motion Study**, the project identified that technical resolution (FCR) and communication empathy are the primary drivers of customer advocacy.

---

## 📈 Key Results
*   **NPS Improvement:** 15.2% ➡️ **28.4%**
*   **Repeat Call Reduction:** 32.4% ➡️ **21.2%**
*   **FCR Improvement:** 67.8% ➡️ **78.5%**
*   **AHT Reduction:** 6.5 Min ➡️ **6.1 Min**
